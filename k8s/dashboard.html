Kubenetes-安装 Dashboard ​

文档

安装 Dashboard ​

sh
kubectl apply -f https://raw.githubusercontent.com/kubernetes/dashboard/v2.7.0/aio/deploy/recommended.yaml

将 ClusterIP 改成 NodePort，保存退出 ​

sh
$ kubectl edit svc kubernetes-dashboard -n kubernetes-dashboard
selector:
  k8s-app: kubernetes-dashboard
sessionAffinity: None
type: ClusterIP

查看 service 访问端口 ​

sh
$ kubectl -n kubernetes-dashboard get service kubernetes-dashboard
NAME                   TYPE       CLUSTER-IP     EXTERNAL-IP   PORT(S)         AGE
kubernetes-dashboard   NodePort   10.97.39.150   <none>        443:31000/TCP   2m47s

查看到service端口为 31000

Dashboard 访问地址 https://172.31.0.10:31000

创建访问账户 ​

sh
vi dashboard-account.yaml

输入以下内容

yaml
apiVersion: v1
kind: ServiceAccount
metadata:
  name: admin-user
  namespace: kubernetes-dashboard

---

apiVersion: rbac.authorization.k8s.io/v1
kind: ClusterRoleBinding
metadata:
  name: admin-user
roleRef:
  apiGroup: rbac.authorization.k8s.io
  kind: ClusterRole
  name: cluster-admin
subjects:
- kind: ServiceAccount
  name: admin-user
  namespace: kubernetes-dashboard

创建账号

sh
kubectl apply -f dashboard-account.yaml

获取访问令牌 ​

sh
# 指定有效期：--duration=315360000s
# 315360000s 代表 10年
kubectl -n kubernetes-dashboard --duration=315360000s create token admin-user
